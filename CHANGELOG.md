# Changelog

## MoonClocks 1.0.0

First public GitHub release build of MoonClocks.

### Added

- Pale Moon and Epyrus compatible legacy XUL extension.
- New MoonClocks name and extension ID: `moonclocks@halvar666`.
- MoonClocks chrome namespace and preference branch.
- OpenStreetMap location links replacing the old Google Earth KML workflow.
- Icons8-based MoonClocks icon with attribution.
- GitHub release notes and repository documentation.

### Changed

- Based on the GPL-licensed FoxClocks 4.2.3 XUL codebase.
- Bundled timezone database carries IANA tzdb 2026b metadata and includes the 2026b British Columbia / America/Vancouver data update.
- Zone Picker cleanup for Kyiv, Ukraine, Crimea, and several newer IANA zones.
- Add-on Manager metadata updated for the MoonClocks fork.

### Removed

- Old upstream post-install/update pages.
- Old Google Earth user-facing links.
- Firefox, Thunderbird, and SeaMonkey target applications from the public MoonClocks install manifest.
