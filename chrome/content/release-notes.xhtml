MoonClocks 1.0.2

MoonClocks is a Pale Moon and Epyrus XUL fork based on the GPL-licensed FoxClocks 4.2.3 codebase.

Changes in this release

  • Fixed the bundled update/release information panel so it shows the current MoonClocks release version.
  • Renamed the main internal install/update/help/release-note helper functions from openFoxClocks* to openMoonClocks*.
  • Renamed the internal version helper from getFoxClocksVersion() to getMoonClocksVersion().
  • Added MoonClocks-named constants for the extension ID, local release-note URLs and settings extension.
  • Kept the old openFoxClocks*, getFoxClocksVersion() and related constants as temporary compatibility aliases.
  • No changes to the bundled timezone database; it remains IANA tzdb 2026b.
  • The manifest-based timezone database update channel from MoonClocks 1.0.1 remains unchanged.

Main MoonClocks changes compared with FoxClocks 4.2.3

  • Renamed fork with a new extension ID: moonclocks@halvar666.
  • Internal chrome URLs use the chrome://moonclocks/ namespace.
  • Runtime preferences use the extensions.moonclocks.* branch.
  • Pale Moon compatibility was updated for current versions.
  • Epyrus support was added.
  • Bundled timezone database was regenerated from IANA tzdb 2026b into the legacy FoxClocks schema 1.2 format.
  • OpenStreetMap is used for location links instead of the old Google Earth integration.
  • Local release notes are used instead of old upstream FoxClocks install/update web pages.
  • MoonClocks 1.0.1 added a manifest-based timezone database update channel.

Important upgrade note

MoonClocks is a separate fork and uses a different extension ID than the original FoxClocks.

Before installing MoonClocks, back up your profile and clock settings. Then uninstall the original FoxClocks extension, or any earlier FoxClocks Legacy / MoonClocks test build, restart the application, and install MoonClocks.

License note

MoonClocks is based on FoxClocks 4.2.3, licensed under GPL-2.0-or-later. The timezone database file was regenerated from the public IANA Time Zone Database. No source files from the WebExtension version of FoxClocks were used.
