MoonClocks 1.0.0

MoonClocks is a Pale Moon and Epyrus XUL fork based on the GPL-licensed FoxClocks 4.2.3 codebase.

Changes in this build

  • Stabilized menubar hover styling in the standalone MoonClocks window on Pale Moon/GTK.
  • Top-level menu labels now keep the same metrics before and during hover, avoiding the visible text jitter seen in Pale Moon.
  • No changes to the timezone database, Zone Picker data, OpenStreetMap integration, or application targets.

Current MoonClocks changes

  • Internal chrome URLs now use the chrome://moonclocks/ namespace.
  • Runtime preferences now use the extensions.moonclocks.* branch.
  • Bundled timezone database regenerated from IANA tzdb 2026b into the legacy FoxClocks schema 1.2 format.
  • OpenStreetMap is used for location links instead of the old Google Earth integration.
  • Local release notes are used instead of upstream FoxClocks install/update web pages.
  • Zone Picker cleanup for Kyiv, Ukraine, Crimea, Simferopol, Famagusta, and selected tzdb additions.
  • Pale Moon and Epyrus target applications are supported.
  • Linux/GTK menubar hover styling adjusted in the standalone MoonClocks window.

Upgrade note

Before moving from the original FoxClocks extension or earlier FoxClocks Legacy test builds to MoonClocks, back up your profile and clock settings, uninstall the older extension, restart the application, and then install MoonClocks.

License note

MoonClocks is based on FoxClocks 4.2.3, licensed under GPL-2.0-or-later. The timezone database file was regenerated from the public IANA Time Zone Database. No source files from the WebExtension version of FoxClocks were used.
